Country Language City

Continents Project

On this website, then you will be introduced to forms, which will let you insert countries, cities and languages.

Use the different menus to get to the different forms, where you underneath will be able to see which items are already in the database. However, you will have to take into account a few restrictions:

Countries can only be added, if the continents are either Antarctica, Africa, Asia, Europe, North America, South America or Oceania.

Languages and cities can only be added, if the country is already added into the database.
